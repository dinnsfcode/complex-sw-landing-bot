# Complex SW landing and Telegram lead bot

Учебный проект: лендинг продукта с формой заявки и Telegram-ботом для обработки лидов.

## Что внутри

- `index.html` - разметка одностраничного лендинга.
- `style.css` - адаптивная верстка, сетки, состояния и визуальная настройка.
- `main.js` - интерактив страницы и отправка формы на backend.
- `img/` - изображения для сайта.
- `bot/` - Telegram-бот на Python/aiogram с приемом заявок и статусами.

## Что реализовано

- Адаптивный лендинг под desktop, планшет и мобильную ширину.
- CTA-кнопки, которые ведут к форме заявки.
- Проверка формы перед отправкой.
- Отправка заявки JSON-запросом на endpoint бота.
- Сохранение заявок в SQLite.
- Меню Telegram-бота для просмотра заявок по статусам.
- Смена статусов: новая, в работе, отказ, выполнено.

## Запуск сайта

Сайт можно открыть как обычный HTML-файл:

```text
index.html
```

Для полной проверки отправки заявки нужен запущенный backend из папки `bot`.

## Запуск бота

1. Перейти в папку `bot`.
2. Установить зависимости:

```bash
pip install -r requirements.txt
```

3. Создать файл `.env` в папке `bot`:

```env
BOT_TOKEN=your_telegram_bot_token
ADMIN_CHAT_ID=your_chat_id
LEAD_SECRET=optional_secret
BOT_WEB_HOST=127.0.0.1
BOT_WEB_PORT=8080
```

4. Запустить:

```bash
python run.py
```

## Важно

Токены, `.env`, локальная база заявок и служебные файлы не входят в репозиторий.

Интеграция сайта и бота в демо-режиме может использовать временный HTTPS-адрес, например ngrok. Для постоянной работы backend нужно разместить на сервере и заменить endpoint в `main.js`.
