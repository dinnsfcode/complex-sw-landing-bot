# Telegram bot

Бот для заявок с лендинга Complex SW СУСТАВЫ.

Токен, ссылка на бота и права доступа передаются отдельно.

Основная структура:

- `run.py` — входная точка бота и HTTP-точки приёма заявок
- `config.py` — настройки из переменных окружения или `.env`
- `database.py` — SQLite-хранилище заявок
- `app/handlers.py` — обработчики команд, меню и статусов
- `app/keyboards.py` — клавиатуры
